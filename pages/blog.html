What is the document object model (DOM)?

The Document Object Model (DOM) is a programming interface that represents and allows manipulation of a webpage's structure, content, and styling as a hierarchical tree of objects.

How do you select an element from the DOM?

To select an element from the DOM, I use DOM selection methods or DOM querying methods. Like document.getElementById(), document.querySelector(), or document.getElementsByClassName(), depending on the element's attributes or selectors.

What is event delegation in the context of DOM, and why it is useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events for multiple child elements, taking advantage of event bubbling. It is useful because it improves performance by reducing the number of event listeners, simplifies code, and allows dynamic elements to be managed without extra setup

How do you manipulate an element's attributes and styles using the DOM?

I manipulate an element's attributes using methods like setAttribute() and removeAttribute(), and its styles by accessing the style property to modify CSS.
